# Security Policy

## Reporting a Vulnerability
We take the security of our project seriously. If you discover a security vulnerability, please do not open a public issue. Instead, please contact the maintainer directly via email.

**Contact:** ilyass.baa0@gmail.com

## Supported Versions
| Version | Supported |
|---------|-----------|
| 1.1.x   | Yes       |
| 1.0.x   | Yes       |

## Security Updates
We strive to keep dependencies updated to avoid known vulnerabilities. All security-related patches will be released as soon as they are tested.
